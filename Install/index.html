Steps to install SCORM Cloud functionality in Dokeos (1.8.6)


  1. Copy your uncompressed scorm_cloud folder into the Dokeos/main folder of your Dokeos system.
  2. Copy the scorm_cloud/lang/english/scorm_cloud.inc.php into the Dokeos/main/lang/english folder (this assumes the installation of Dokeos uses English as the primary language).
  3. Run the scorm_cloud/db/scorm_cloud.sql script on your main Dokeos mysql database.
  4. Ensure that the scorm_cloud/uploads folder is writeable by the system.
  5. Modify the following files with the included code blocks. If you are installing this on top of a 1.8.6 release of Dokeos, and you have not customized the following files, you can instead replace the following files with the modified versions located in the scorm_cloud/install/core_files folder.  (It is recommended that any time you overwrite files in this manner, you first back up the originals.)
